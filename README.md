# Lung Cancer Survival Prediction (Data Mining)

This project uses a Random Forest Classifier to predict the survival status of lung cancer patients based on clinical data.

## 📊 Dataset
The dataset used is `Lung Cancer(100k)(Classification).csv`, which contains 100,000 records of patient data including BMI, smoking history, and treatment types.

## 🛠️ Features
- **Data Preprocessing:** Scaling of numerical variables and train-test splitting.
- **EDA:** Visualizations for survival distribution and feature correlation.
- **Model:** Random Forest Classifier.
- **Evaluation:** Accuracy, Confusion Matrix, and F1-Score analysis.

## 📋 Requirements
To run this locally, you will need:
- Python 3.x
- Pandas
- Scikit-learn
- Seaborn
- Matplotlib
